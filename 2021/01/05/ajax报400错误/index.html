$.ajax 报400错误

今天在实际的项目中将jQuery的$.ajax请求方式由get改为post，发现报400错误；原因是由于contentType设置错误

Get
Post

默认不设置也没问题

Java

java后端@RequestMapping的Method保持默认设置即可，可接受GET、POST、PUT、DELETE等请求。
