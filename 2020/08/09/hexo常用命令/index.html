Hexo常用的几个命令

1. hexo server

hexo server ，可简写为 hexo s，启动本地服务器预览，默认地址为http://localhost:4000

2. hexo new

hexo new 创建一篇新的文章，后接文章名。

3. hexo generate

hexo generate，可简写为hexo g，生成静态文件。

4. hexo clean

hexo clean 清除缓存文件 (db.json) 和已生成的静态文件 (public)。

5. hexo deploy

hexo deploy ，可简写为hexo d，文件生成后立即部署网站

详细可访问Hexo官网查看更多命令。
